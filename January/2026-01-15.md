# chojoonhee-til

## 날짜: 2026-01-15

### 스크럼
- 커뮤니티 백엔드: 게시물 관련 API(게시물 작성/삭제/수정 등) 작성 완료 후 2차 피드백을 받고 반영.
    - 게시물 작성, 삭제, 수정
    - 게시물 목록 페이지
    - 게시물 좋아요, 조회수, 댓글 수
    - 댓글 작성, 삭제, 수정
    - 사용자 모델, 로그인/로그아웃, 세션 관리 기능 보완
- 커뮤니티 백엔드: 유틸리티 모듈을 만들어 독립적으로 쓰이는 기능을 분리해 기존 컨트롤러 함수들을 단순화.

### 새로 배운 내용
#### API 개발: 실전 지식
- `DELETE` 메소드가 정상적으로 실행되었을 때, 204로 응답하도록 설정하는 이유
- FastAPI의 정책과 프로그램 구조의 관계
- 라우터와 컨트롤러의 역할
- 서비스 계층의 역할
- 스키마 계층의 역할

#### 파이썬: 라이브러리에서 사용하는 심화 문법들
- `replace()`
- `Optional[str]`과 `str | None`의 차이
