- B-Tree
- B+ Tree
- 인덱스 페이지
- 클러스터형 인덱스, 비클러스터형 인덱스
- Cardinality
- Selectivity
- 페이지 분할

- ngram 파서

- SQL EXPLAIN
- Index Scan
- Full Table Scan

#### Transaction: 데이터베이스 연산의 논리적 단위
- 트랜잭션의 상태
    - Active
    - Partially Commited
    - Commited
    - Aborted
    - Failed
- 롤백
- ACID
    - Atomicity
    - Consistency
    - Isolation
    - Durability
- Isolation Level
    - Read Uncommited
    - Read Commmited
    - Repeatable Read
    - Serializable
- SQL로 트랜잭션을 정의하는 방법
- 데이터 읽기 이상 현상
    - Dirty Read
    - Non-Repeatable Read
    - Phantom Read
