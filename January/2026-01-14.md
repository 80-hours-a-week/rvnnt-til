### 새로 배운 내용
#### JWT: JSON Web Token
- JWT의 기본 구조 (Header, Payload, Signature)
- JWT를 실제로 사용하는 방법
- JWT + 쿠키/세션
- SPA 구조

#### OAuth: 소셜 계정을 통한 회원가입과 로그인
- OAuth의 기본 개념
- OAuth의 인증 절차
- OAuth를 실제로 사용하는 방법

#### HTTPS: HTTP Secure
- HTTP 자격증명 헤더
    - 기본 인증
    - 다이제스트 인증
    - 베어러 인증
- HTTP와 HTTPS의 차이
- HTTPS의 통신 과정
    - SSL/TLS
    - ECDHE
        - PFS
    - Certificate Authority (CA)

#### 네트워크: 구석구석 숨어있는 심화 개념들
- Loopback Interface
- Broadcast Domain
- Trunking
- Virtual Network Interface Card (VNIC)
- Overlay Network & Underlay Network
- Penetration Test
