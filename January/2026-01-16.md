### 새로 배운 내용
#### 스트리밍 서비스: 스트리밍 서비스에서 지연시간·대역폭·전송 안정성을 좌우하는 핵심 요소
- VOD
- 라이브 스트리밍
- 초저지연 서비스
- 지연시간
    - 단방향 지연
    - Round-Trip Time(RTT)
    - 전파 지연(Propagation Delay)
    - 전송 지연(Transmission Delay / Serialization Delay)
    - 처리 지연(Processing Delay)
    - 큐잉 지연(Queuing Delay)
    - 버스트
- 대역폭
    - 처리량(Throughput)
    - Protocol Window
    - Goodput
    - Shannon-Hartley Theorem
    - 누화(Crosstalk)
- 안정성
    - Jitter
    - 네트워크의 오류 복구 전략
        - Automatic Repeat Request(ARQ)
        - Forward Error Correction(FEC)

#### VOD 아키텍처: 품질과 효율성의 극대화
- Multi-pass Encoding
    - Encoding Ladder
    - Adaptive Bitrate Streaming(ABR)
- 계층화된 저장소
    - Hot / Cold
- Edge CDN
    - Anycast
    - Remote Direct Memory Access
    - Edge Network
    - Zero-copy
    - Agressive Buffering

#### 라이브 스트리밍 아키텍처: 균형점 찾기
- Ingest
    - First Mile, Last Mile
    - Glass-to-Glass Latency
- Common Media Application Format(CMAF)
- Flash Crowd / Thundering Herd
- 요청 병합(Request Coalescing)
- Longtail Caching

#### 초저지연 서비스 아키텍처: 물리적 한계에 도전하다
- Interactive Connectivity Establishment(ICE)
- 적시성(Timeliness)
- Timing Wheel Algorithm
    - Pacing
- 패킷 분류 알고리즘
    - Grid of Tries
    - Recursive Flow Classification(RFC)
